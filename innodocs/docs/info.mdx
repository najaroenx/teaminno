# <p className="text-4xl font-bold"> DLT API</p>

 <h2 className="text-xl font-bold ">Features</h2>

<div className="my-8">
<p className="text-gray-700 leading-7 py-4">
  แพลตฟอร์มแต้มบนบล็อกเชนสำหรับร้านค้า (Merchant) และลูกค้า (Customer) ใช้งานง่ายแต่ปลอดภัย
  รองรับตั้งแต่การออกแต้ม โอนแต้ม จนถึงดูประวัติธุรกรรม
</p>

<div className="grid md:grid-cols-1 gap-4">
  <div className="rounded-xl border border-gray-200 p-4 bg-white">
    <h3 className="text-xl font-semibold text-gray-70 ">สำหรับผู้ใช้งาน/ธุรกิจ (Non-Developer)</h3>
    <ul className="list-disc ml-5 mt-2 space-y-1">
      <li><b>สร้างร้านค้าและกระเป๋าแต้มอัตโนมัติ</b> – เปิด Merchant ใหม่ ระบบจะสร้าง <b>Wallet</b> ให้ทันที</li>
      <li><b>ออกแต้ม/เติมแต้มให้ลูกค้าได้</b> – ใช้แต้มเป็นรางวัล/โปรโมชัน (เช่น สมัครสมาชิกใหม่ รับ 150 แต้ม)</li>
      <li><b>ลูกค้าโอนแต้มให้กันได้</b> – ส่งแต้มระหว่างลูกค้าด้วยกัน (C2C)</li>
      <li><b>ดูยอดคงเหลือและประวัติธุรกรรม</b> – เช็กยอดย้อนหลังตามช่วงเวลา</li>
      <li><b>ความปลอดภัย</b> – ใช้ <code>API Key</code>, ธุรกรรมบันทึกบนบล็อกเชน, <code>Idempotency-Key</code></li>
      <li><b>ตรวจสถานะธุรกรรมแบบเรียลไทม์</b> – `PENDING → CONFIRMED/FAILED`</li>
      <li><b>รองรับการขยายระบบ</b> – REST API ใช้งานกับ CRM/POS/Mobile app ได้ง่าย</li>
    </ul>
  </div>

  <div className="rounded-xl border border-gray-200 p-4 bg-white">
    <h3 className="text-xl font-semibold text-gray-700">สำหรับนักพัฒนา (Developer)</h3>
    <div className="text-sm text-gray-700 mt-2">
      <p><b>Base URL</b>: <code>https://api.example.com</code></p>
      <p><b>Auth</b>: <code>Authorization: Bearer &lt;API_KEY&gt;</code></p>
      <p><b>Idempotency</b> (write ops): <code>Idempotency-Key: &lt;unique-key&gt;</code></p>
    </div>
  </div>
